# Manual de Usuario

## Introducción
Este manual describe cómo utilizar el sistema desde la perspectiva del usuario final.

## Acceso al sistema
1. Abrir la aplicación
2. Ingresar credenciales
3. Acceder al panel principal

## Funcionalidades principales
- Consultar información
- Crear registros
- Editar registros
- Eliminar registros
- Descargar reportes

## Recomendaciones
- No compartir credenciales
- Verificar datos antes de guardar
- Cerrar sesión al terminar
